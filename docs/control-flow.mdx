# Управляющие конструкции

Управляющие конструкции позволяют управлять ходом выполнения программы. IdzeyKL поддерживает различные условные операторы и циклы.

## Условный оператор if-else

Синтаксис:
```
if (условие) {
    // Код, выполняемый, если условие истинно
} else {
    // Код, выполняемый, если условие ложно
}
```

Пример:
```
var age = 18;
if (age >= 18) {
    println("Совершеннолетний");
} else {
    println("Несовершеннолетний");
}
```

Также поддерживается сокращенная форма без блока else:
```
if (условие) {
    // Код, выполняемый, если условие истинно
}
```

### Вложенные if

Можно вкладывать if друг в друга для сложных условий:
```
var age = 20;
var hasLicense = true;

if (age >= 18) {
    if (hasLicense) {
        println("Может водить");
    } else {
        println("Нужно получить права");
    }
} else {
    println("Слишком молод для вождения");
}
```

## Цикл loop (1 вариант синтаксиса)

Синтаксис:
```
loop (инициализация; условие; инкремент) {
    // Тело цикла
}
```

Пример:
```
loop (var i = 0; i < 5; i = i + 1) {
    println(i);
}
```

## Цикл loop (2 вариант синтаксиса)

Синтаксис:
```
loop (условие) {
    // Тело цикла
}
```

Пример:
```
var i = 0;
loop (i < 5) {
    println(i);
    i = i + 1;
}
```

### Операторы управления циклом

#### break

Оператор `break` немедленно завершает цикл:
```
for (var i = 0; i < 10; i = i + 1) {
    if (i == 5) {
        break;
    }
    println(i);
}
```

#### continue

Оператор `continue` пропускает текущую итерацию и переходит к следующей:
```
for (var i = 0; i < 5; i = i + 1) {
    if (i == 2) {
        continue;
    }
    println(i);
}
```

## Типовые шаблоны

### Счётчик вверх
```
for (var i = 0; i < n; i = i + 1) {
    // Код
}
```

### Счётчик вниз
```
for (var i = n - 1; i >= 0; i = i - 1) {
    // Код
}
```

### Перебор массива
```
var numbers = {1, 2, 3, 4, 5};
for (var i = 0; i < numbers.length; i = i + 1) {
    println(numbers[i]);
}
```

### Условный цикл
```
var found = false;
var i = 0;
while (!found && i < array.length) {
    if (array[i] == target) {
        found = true;
    }
    i = i + 1;
}
```

## Рекомендации

1. **Используйте осмысленные условия**:
   ```
   // Хорошо
   if (age >= 18) { ... }
   
   // Плохо
   if (age > 17) { ... }
   ```
2. **Избегайте глубокой вложенности**:
   ```
   // Хорошо
   if (!isValid) {
       return;
   }
   // Основная логика
   
   // Плохо
   if (isValid) {
       // Основная логика
   }
   ```
3. **Используйте подходящий тип цикла**:
   - `for` — когда известно количество итераций
   - `while` — когда количество итераций заранее неизвестно
4. **Инициализируйте переменные до цикла**:
   ```
   // Хорошо
   var sum = 0;
   for (var i = 0; i < n; i = i + 1) {
       sum = sum + i;
   }
   ```
