## Операторы

### Арифметические операторы
- `+` : Сложение (также используется для конкатенации строк)
- `-` : Вычитание
- `*` : Умножение
- `/` : Деление
- `%` : Остаток от деления

### Операторы сравнения
- `==` : Равно
- `!=` : Не равно
- `<` : Меньше
- `<=` : Меньше или равно
- `>` : Больше
- `>=` : Больше или равно

### Логические операторы
- `&&` : Логическое И
- `||` : Логическое ИЛИ
- `!` : Логическое НЕ

### Оператор присваивания
- `=` : Присваивание

### Унарные операторы
- `-` : Унарный минус
- `!` : Логическое отрицание
