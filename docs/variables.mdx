# Переменные

Переменные объявляются с помощью ключевого слова `var`, за которым следует имя переменной и, опционально, начальное значение.

Синтаксис:
```
var имя_переменной;
var имя_переменной = значение;
```

Примеры:
```
var x;           // Объявление переменной без инициализации (значение null)
var y = 10;      // Объявление переменной с инициализацией
var name = "John"; // Строковая переменная
var isActive = true; // Логическая переменная
var numbers = {1, 2, 3}; // Массив
```

# Операторы

## Арифметические операторы
- `+` : Сложение (также используется для конкатенации строк)
- `-` : Вычитание
- `*` : Умножение
- `/` : Деление
- `%` : Остаток от деления

## Операторы сравнения
- `==` : Равно
- `!=` : Не равно
- `<` : Меньше
- `<=` : Меньше или равно
- `>` : Больше
- `>=` : Больше или равно

## Логические операторы
- `&&` : Логическое И
- `||` : Логическое ИЛИ
- `!` : Логическое НЕ

## Оператор присваивания
- `=` : Присваивание

## Унарные операторы
- `-` : Унарный минус
- `!` : Логическое отрицание

## Приоритет операторов

Операторы вычисляются в следующем порядке (от высшего к низшему):

1. Унарные операторы (`-`, `!`)
2. Умножение, деление, остаток (`*`, `/`, `%`)
3. Сложение, вычитание (`+`, `-`)
4. Операторы сравнения (`<`, `<=`, `>`, `>=`)
5. Операторы равенства (`==`, `!=`)
6. Логическое И (`&&`)
7. Логическое ИЛИ (`||`)
8. Операторы присваивания (`=`, `+=`, `-=`, и т.д.)

Вы можете использовать скобки `()` для изменения порядка вычислений:
```ikl
var result = (2 + 3) * 4; // 20, а не 14
```
